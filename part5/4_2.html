AJAX-POST

  • jquery中函数post()的语法如下
$.post('请求地址url',{参数},function(data){
    //回调函数处理
    //使用data接收返回的json数据
});
  • 这个方法以http协议的post方式请求指定的url地址

代码示例

  • 在d1_ajax中定义视图stu_add，添加学生信息，并返回json数据
from flask import request

@app.route('/stu/add', methods=['POST'])
def stu_add():
    # 接收学生姓名、班级编号
    dict1 = request.form
    stu_name = dict1.get('name')
    class_id = dict1.get('class_id')
    # 创建学生对象
    stu = Student()
    stu.name = stu_name
    stu.class_id = int(class_id)
    # 保存到数据表
    db.session.add(stu)
    db.session.commit()
    # 返回结果为用户的编号
    return jsonify(result=stu.id)
  • 在d1_index.html中使用$.post()发起请求，写在“添加按钮的点击处理代码”处
var stu_name = '张三';
var class_id = '1';
$.post('/stu/add', {
    'name': stu_name,
    'class_id': class_id
}, function (data) {
    vue_tb_list.stu_list.push({'id': data.result, 'name': stu_name});
});

ajax中的CSRF验证

  • 注意：在post请求时需要进行CSRF口令验证
  • 修改d1_ajax.py文件，添加CSRF保护
from flask_wtf.csrf import CSRFProtect

app.config['SECRET_KEY'] = 'itheima'

CSRFProtect(app)
